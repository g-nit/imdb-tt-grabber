# IMDB TT Grabber

**IMDB TT Grabber** is a simple Chrome extension that extracts the IMDB title from any IMDB URL and makes it easy to copy. Perfect for quickly grabbing movie or TV show titles for spreadsheets, notes, or social sharing.
